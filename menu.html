Para esta página decidí hacer el prototipo de una web que funcione como interfaz para administrar los servicios que ofrece la sala de computo de la biblioteca de FIME, donde realice mi servicio social. Para el funcionamiento completo de una interfaz como esta necesitaría implementar otros lenguajes como Javascript o PHP pero para este modulo desarrolle la parte de HTML y CSS. La pagina cuenta con una pantalla de registro y otra de login desde donde se puede acceder al menú principal en que hay una barra lateral que lleva a otras pantallas que realizarían distintas operaciones. La razón de haber desarrollado esta idea es por que la interfaz que se utiliza actualmente en la sala de computo de FIME es algo compleja y quise simplificarla mediante lo que aprendí el módulo para tratar de ayudar a mis ex-compañeros e intentar realizar un proyecto bajo las especificaciones que ellos me dieran como usuarios.

*Dentro de cada opción del menú principal:
- En la barra lateral derecha se encuentra mas información sobre cual es la idea de esa pantalla.
- Pulsa el botón “Menú principal” que aparece al fondo de la barra lateral para volver a esta pantalla.
